C

Claude Code

Anthropic 官方客户端

01 安装

Terminal 
npm install -g @anthropic-ai/claude-code

02 配置 (settings.json)

~/.claude/settings.json 
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY",
    "ANTHROPIC_BASE_URL": "https://www.mhcoding.xyz/"
  }
}

03 启动

claude
O

Codex CLI

OpenAI 官方客户端

注意：Codex 需要使用 Node 22+ 及特定的专属 API Key。

01 安装

Terminal 
npm install -g @openai/codex

02 配置 (config.toml)

~/.codex/config.toml 
[model_providers.mhcoding]
name = "mhcoding"
base_url = "https://www.mhcoding.xyz/v1"
wire_api = "responses"
requires_openai_auth = true

03 凭证 (auth.json)

~/.codex/auth.json 
{ "OPENAI_API_KEY": "YOUR_KEY" }
G

Gemini CLI

Google 官方客户端

01 安装

Terminal 
npm install -g @google/gemini-cli

02 环境 (.env)

~/.gemini/.env 
GOOGLE_GEMINI_BASE_URL=https://www.mhcoding.xyz/
GEMINI_API_KEY=YOUR_KEY
GEMINI_MODEL=gemini-3-pro

03 启动 (settings.json)

~/.gemini/settings.json 
{
  "security": {
    "auth": { "selectedType": "gemini-api-key" }
  }
}
